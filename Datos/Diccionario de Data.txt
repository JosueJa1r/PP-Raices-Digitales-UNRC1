Anio  == Año 

Idestado == ID de estado 

Nomestado == Nombre de estado 

Idddr  == ID Distrito de Desarrollo Rural

Nomddr == Distrito de Desarrollo Rural

Idcader	== ID nombre del Centro de Apoyo al Desarrollo Rural

Nomcader == nombre del Centro de Apoyo al Desarrollo Rural

Idmunicipio  ==	ID Munucipio

Nommunicipio == Nombre de Municipio

Idciclo	== ID temporada de siembra

Nomcicloproductivo == temporada de siembra

Idmodalidad == ID Indica la fuente de agua del cultivo. 
Puede ser Riego //[1] (si cuenta con sistemas de agua) o 
Temporal //[2](si depende únicamente de las lluvias).

Nommodalidad == Indica la fuente de agua del cultivo. 
Puede ser Riego //[1] (si cuenta con sistemas de agua) o 
Temporal //[2](si depende únicamente de las lluvias).

Idunidadmedida == ID Define en qué se mide la cosecha, 
Tonelada, Planta, Gruesa o Manojo

Nomunidad == ID Define en qué se mide la cosecha, 
Tonelada, Planta, Gruesa o Manojo

Idcultivo == ID producto agrícola

Nomcultivo == producto agrícola

Sembrada  == La superficie total de tierra en la que se colocó la semilla o planta.

Cosechada == La superficie de la cual se pudo levantar la cosecha con éxito.

Siniestrada == Superficie que se perdió totalmente debido a factores climáticos, plagas o desastres.

Volumenproduccion == La cantidad total producida en las unidades indicadas en Nomunidad.

Rendimiento == Es la eficiencia del campo; indica cuánto se produjo por cada unidad de superficie cosechada.

Precio == El "Precio Medio Rural", que es el pago promedio por unidad que recibió el agricultor al vender su producto

Valorproduccion == Es el valor económico total de la cosecha, calculado multiplicando el volumen total por el precio de venta.
